# 🏥 IntelliHealth :An AI Powered Diabetes Prediction System

## 📋 Overview
Machine learning application for predicting diabetes risk using patient health data with intelligent risk categorization.

## 📊 Dataset
**Pima Indians Diabetes Dataset** (UCI Machine Learning Repository / Kaggle)

## 🛠️ Tech Stack
```
Python • Pandas • NumPy • Scikit-learn • Streamlit
```

## ✨ Key Features
- ✅ **Input Validation**: Checks for negative values
- 📈 **Probability Scores**: Risk categorization with probability percentages
  - 🟢 **Low Risk** (< 30% probability)
  - 🟡 **Medium Risk** (30-60% probability)
  - 🔴 **High Risk** (> 60% probability)
- 🔍 **Explainability**: Shows top contributing factors

## 🎯 Model Performance
- **Algorithm**: SVM Classifier
- **Test Accuracy**: 81.8%
- **Features**: 8 health parameters
